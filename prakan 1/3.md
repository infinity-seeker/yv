> भरद्वाज उवाच

#### 1
```
जीवन्मुक्त-स्थितिम् ब्रह्मन् कृत्वा राघवम् आदित: । 
क्रमात् कथय मे नित्यम् भविष्यामि सुखी यथा ॥१।
```

#### 2
```
भ्रमस्य जागतस्य_अस्य जातस्य_आकाश-वर्णवत् ।
अपुन: स्मरणम् मन्ये साधो विस्मरणम् वरम् ॥१।३।२॥
```

#### 3
```
दृश्य-अत्यन्त-अ.भाव-बोधम् विना तन्_न_अनुभूयते । 
कदा.चित्_केन.चिन्_नाम स्व.बोधो_अन्विष्यताम् अत: ॥१।३।३॥
```

#### 4
```
स च_इह संभवत्य्_एव तद्_अर्थम् इदम् आततम् । 
शास्त्रम् आकर्णयसि चेत् तत्त्वम् आप्स्यसि न_अन्यथा ॥१।३।४॥ 
```

#### 5
```
जगद्-भ्रमो ऽयम् दृश्यो ऽपि न_अस्त्य्_एव_इत्य्_अनुभूयते ।
वर्णो व्योम्न इव_अखेदाद्_विचारेन_अमुना_अनघ ॥१।३।५॥
```

#### 6
```
दृश्यम् न_अस्ति_इति बोधेन मनसो दृश्य-मार्जनम् ।
सम्पन्नम् चेत् तद्_उत्पन्ना परा निर्वाण-निर्वृति: ॥१।३।६॥
```

#### 7
```
अन्यथा शास्त्र-गर्तेषु लुठताम् भवताम् इह । 
भवत्य्_अकृत्रिम~अज्ञानम् कल्पैर्_अपि न निर्वृति: ॥१।३।७॥
```

#### 8
```
अशेषेण परित्यागो वासनानाम् य उत्तम: ।
मोक्ष इत्य्_उच्यते ब्रह्मन् स एव विमल-क्रम: ॥१।३।८॥
```

#### 9
```
क्षीणायाम् वासनायाम् तु चेतो गलति सत्वरम् ।
क्षीणायाम् शीत-संतत्याम् ब्रह्मन् हिम-कणो यथा ॥१।३।९॥
```

#### 10
```
अयम् वासनया देहो ध्रियते भूत-पञ्जर: । 
तनुना_अन्तर्-निविष्टेन मुक्त~ओघस्_तन्तुना यथा ॥१।३।१०॥
```

#### 11
```
वासना द्वि.विधा प्रोक्ता शुद्धा च मलिना तथा । 
मलिना जन्मनो हेतु: शुद्धा जन्म-विनाशिनी ॥१।३।११॥ 
```

#### 12
```
अ.ज्ञान-सु.घन-आकारा घन-अहम्.कार-शालिनी । 
पुनर्-जन्म-करी प्रोक्ता मलिना वासना बुधै: ॥१।३।१२॥
```

#### 13
```
पुनर्-जन्म-अङ्कुरम् त्यक्त्वास्थिता सम्भृष्ट-बीजवत् । 
देह~अर्थम् ध्रियते ज्ञात-ज्ञेया शुद्धा_इति च_उच्यते ॥१।३।१३॥
```

#### 14
```
अपुनर्जन्म-करणी जीवन्मुक्तेषु देहिषु ।
वासना विद्यते शुद्धा देहे चक्र_ इव भ्रम: ॥१।३।१४॥
```

#### 15
```
ये शुद्ध-वासना भूयो न जन्म~अनर्थ-भाजनम् ।
ज्ञात-ज्ञेयास् त उच्यन्ते जीवन्मुक्ता महा-धिय: ॥१।३।१५॥ 
```

#### 16
```
जीवन्मुक्ति-पदम् प्राप्तो यथा रामो महा-मति: ।
तत् ते ऽहम् शृणु वक्ष्यामि जरा-मरण-शान्तये ॥१।३।१६॥
```

#### 17
```
भरद्वाज महा-बुद्धो राम-क्रमम् इमम् शुभम् ।
शृणु वक्ष्यामि तेन_एव सर्वम् ज्ञास्यसि सर्वदा ॥१।३।१७॥
```

#### 18
```
विद्या-गृहाद्_विनिष्क्रम्य रामो राजीव.लोचन: ।
दिवसान्य्_अनयद्_गेहे लीलाभिर्_अकुतो-भय: ॥१।३।१८॥
```

#### 19
```
अथ गच्छति काले तु पालयत्य्_अवनिम् नृपे ।
प्रजासु वीत.शोकासु स्थितासु विगत.ज्वरम् ॥१।३।१९॥
```

#### 20
```
तीर्थ-पुण्य~आश्रम-श्रेणीर्_द्रष्टुम् उत्काण्ठितम् मन: ।
रामस्य_अभूद्_भृशम् तत्र कदाचिद्_गुण-शालिन: ॥१।३।२०॥ 
```

#### 21
```
राघवश्_चिन्तयित्वा_एवम् उपेत्य चरणौ पितु: ।
हंस: पद्माव्_इव नवौ जग्राह नख-केसरौ ॥१।३।२१॥
```

#### 22
```
तीर्थानि देव-सद्मानि वनान्य्_आयतनानि च ।
द्रष्टुम् उत्कण्ठितम् तात मम_इदम् नाथ मानसम् ॥१।३।२२॥
```

#### 23
```
तद्_एताम् अर्थिताम् पूर्वाम् स.फलाम् कर्तुम् अर्हसि ।
न सो_अस्ति भुवने नाथ त्वया यो_अर्थी न मानित: ॥१।३।२३॥
```

#### 24
```
इति संप्रार्थितो राजा वसिष्ठेन समम् तदा ।
विचार्यामुञ्चद् एवैनम् रामम् प्रथमम् अर्थिनम् ॥१।३।२४॥
```

#### 25
```
शुभे नक्षत्र-दिवसे भ्रातृभ्याम् सह राघव: ।
मङ्गल-अलम्.कृत-वपु: कृत-स्वस्त्य्~अयनो द्विजै: ॥१।३।२५॥
```

#### 26
```
वसिष्ठ-प्रहितैर्_विप्रै: शास्त्र-ज्ञैश्_च समन्वित: ।
स्निग्धै: कतिपयैर्_एव राज-पुत्र-वरै: सह ॥१।३।२६॥
```

#### 27
```
अम्बाभिर्_विहित-अशीर्भिर्_आलिङ्ग्य-आलिङ्ग्य भूषित: ।
निरगात्_स्व.गृहात् तस्मात्_तीर्थ-यात्र-अर्थम् उद्यत: ॥१।३।२७॥
```

#### 28
```
निर्गत: स्व-पुरात्_पौरेस्_तूर्य-घोषेण वादित: ।
पीयमान: पुर-स्त्रीणाम् नेत्रैर्_भृङ्ग~ओघ-भंगुरै: ॥१।३।२८॥
```

#### 29
```
ग्रामीण-ललनालोल-हस्त-पद्म~आपन-उदितै: ।
लाज-वर्षैर्_विकीर्ण~आत्मा हिमैर्_इव हिम~अचल: ॥१।३।२९॥
```

#### 30
```
आवर्जयन् विप्र-गणान् परिशृण्वन् प्रजाशिष: ।
आलोकयन् दिगन्तांश्_च परिचक्राम जाङ्गलान् ॥१।३।३०॥
```

#### 31
```
अथ_आरभ्य स्वकात् तस्मात्_क्रमात्_कोशल-मण्डलात् ।
स्नान-दान-तपो-ध्यान-पूर्वकम् स ददर्श.ह ॥१।३।३१॥
```

#### 32
```
नदी-तीराणि पुण्यानि वनान्य्_आयतनानि च ।
जङ्गलानि जनान्तेषु तटान्य्_अब्धि-मही.भृताम् ॥१।३।३२॥
```

#### 33
```
मन्दाकिनीम् इन्दु-निभाम् कालिन्दीम् च_उत्पल~अमलाम् ।
सरस्वतीम् शतद्रूम् च चन्द्र-भागाम् इरावतीम् ॥१।३।३३॥
```

#### 34
```
वेणीम् च कृष्ण-वेणीम् च निर्विन्ध्याम् सरयूम् तथा ।
चर्मण्वतीम् वितस्ताम् च विपाशाम् बाहुदाम् अपि ॥१।३।३४॥
```

#### 35
```
प्रयागम् नैमिषम् च_एव धर्म~अरण्यम् गयाम् तथा ।
वाराणसीम् श्रीगिरिम् च केदारम् पुष्करम् तथा ॥१।३।३५॥
```

#### 36
```
मानसम् च क्रम-सरस्_तथा_एव_उत्तर-मानसम् ।
वडव-अवदनम् च_एव तीर्थ-वृन्द च सागरम् ॥१।३।३६॥
```

#### 37
```
अग्नितीर्थम् महातीर्थम् इन्द्रद्युम्न-सरस्_तथा ।
सरांसि सरितश्_च_एव तथा नद-ह्रद-अवलीम् ॥१।३।३७॥
```

#### 38
```
स्वामिनम् कार्तिकेयम् च शालग्रामम् तथा ।
स्थानानि च चतु:षष्टिम् हरेर्_अथ हरस्य च ॥१।३।३८॥
```

#### 39
```
नाना-आश्चर्य-विचित्राणि चतुर्-अब्धि-तटानि च ।
विन्ध्य-मन्दर-कुञ्जांश्_च कुल-शैल-स्थलानि च ॥१।३।३९॥
```

#### 40
```
राजर्षीणाम् च महताम् ब्रह्मार्षीणाम् तथा_एव च ।
देवाणाम् ब्रह्मणानाम् च पावनान् आश्रमाञ्_छुभान् ॥१।३।४०॥
```

#### 41
```
भूयो~भूय: स बभ्राम भ्रातृभ्याम् सह मानद: ।
चतुर्ष्व्_अपि दिग्~अन्तेषु सर्वान्_एव मही-तटान् ॥१।३।४१॥ 
```

#### 42
```
अमर-किन्नर-मानवम् आनित:
समवलोक्य महीम् अखिलाम् इमाम् ।
उपययौ स्वगृहम् रघु-नन्दनो
विहृत-दिक्-शिव-लोकम् इव_ईश्वर: ॥१।३।४२॥
```
